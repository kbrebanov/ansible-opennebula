opennebula
==========

Installs and configures OpenNebula

Requirements
------------

This role requires Ansible 1.4 or higher.

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

Install OpenNebula
```
- hosts: all
  roles:
    - { role: kbrebanov.opennebula }
```
